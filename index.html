Four-Layer Sovereign Architecture
VIEW FULL SOURCE
C++20 Recalculation Engine
sovereign_engine.cpp

Zero-copy memory via std::span. Lock-free speculative reevaluation using std::atomic_ref + CAS with thread ID tie-breakers. Bypasses .NET GC entirely.

C# FFI + OLAP Bridge
EngineInterop.cs • PivotEngine.cs

P/Invoke to native engine. DuckDB.NET delivers O(1) columnar aggregations over local SQLite WAL. Designed for 10k+ row clinical asset datasets.

Direct Thermal Printing
GiaiPrintService.cs

Direct System.Drawing.Printing to Zebra ZT411 and compatible devices. Bypasses all dialogs for true clinical-speed GS1 GIAI (8004) label output.

Canvas2D Executive UI
univer_executive.html

High-performance 100,000+ row grid powered by UniverJS Canvas2D. Real-time OLAP, GIAI validation, batch rendering mode, and full bilingual support — all inside WebView2.

Build from Source

Fully reproducible. Minimal dependencies. Designed for air-gapped Windows/Linux environments.

C++20 Engine
WINDOWS (MSVC)
cl.exe /O2 /LD /std:c++20 sovereign_engine.cpp /link /out:sovereign_engine.dll
LINUX (GCC)
g++ -O3 -shared -fPIC -std=c++20 sovereign_engine.cpp -o libsovereign_engine.so
C# Desktop Application
PLACE NATIVE BINARY THEN RESTORE
dotnet add package Microsoft.Data.Sqlite
dotnet add package DuckDB.NET.Data

dotnet run
Copy sovereign_engine.dll (or .so) into the output directory.

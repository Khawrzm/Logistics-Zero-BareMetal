Four-Layer Open-Source Architecture
VIEW FULL SOURCE
C++20 Recalculation Engine
sovereign_engine.cpp

Zero-copy memory via std::span. Lock-free speculative reevaluation using std::atomic_ref + CAS with thread ID tie-breakers. Bypasses .NET GC entirely.

C# FFI + OLAP Bridge
EngineInterop.cs • PivotEngine.cs

P/Invoke to native engine. DuckDB.NET delivers O(1) columnar aggregations over local SQLite WAL. Supports executing arbitrary SQL queries on local vault data.

Direct Label Printing
GiaiPrintService.cs

Direct System.Drawing.Printing to thermal label devices. Bypasses all dialogs for true clinical-speed GS1 GIAI (8004) label output.

Canvas2D Grid UI
univer_executive.html

High-performance 100,000+ row grid powered by UniverJS Canvas2D. Real-time OLAP, GIAI validation, batch rendering mode, and full bilingual support — all inside WebView2.

Sovereign by Architecture

Every decision eliminates external dependencies and maximizes institutional sovereignty.

True Air-Gapped Operation
Zero network calls after deployment. No telemetry, no license checks, no cloud components.
Direct Hardware Control
Thermal printers addressed at the lowest Windows level. No spoolers, no intermediate services.
Local Data Sovereignty
SQLite + DuckDB WAL. All data remains on-premise. Built for PDPL and institutional compliance.
Lock-Free Speculative Engine
C++20 atomics eliminate traditional locks. Cyclic dependencies resolved without deadlocks or mutexes.
Bilingual Interface
Full English + Arabic RTL support with proper cultural formatting for KSA healthcare environments.
Minimal Trust Surface
Verifiable from-source build. No third-party runtimes beyond the .NET 8 desktop runtime.

Build from Source

Fully reproducible. Minimal dependencies. Designed for air-gapped Windows/Linux environments.

C++20 Engine
WINDOWS (MSVC)
cl.exe /O2 /LD /std:c++20 sovereign_engine.cpp /link /out:sovereign_engine.dll
LINUX (GCC)
g++ -O3 -shared -fPIC -std=c++20 sovereign_engine.cpp -o libsovereign_engine.so
C# Desktop Application
PLACE NATIVE BINARY THEN PUBLISH
dotnet add package Microsoft.Data.Sqlite
dotnet add package DuckDB.NET.Data

dotnet publish -c Release -r win-x64 --self-contained true
